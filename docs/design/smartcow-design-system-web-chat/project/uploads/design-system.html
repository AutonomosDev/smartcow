Tokens mobile — patrón obligatorio en todas las screens
Fuentes — inline en cada screen (NO importar desde packages/tokens)

  regular: 'DMSans_400Regular',
  medium:  'DMSans_500Medium',
  bold:    'DMSans_600SemiBold',
};
Declarar al tope de cada archivo .tsx. El import desde packages/tokens/theme NO existe en el proyecto mobile — causa error de build.
